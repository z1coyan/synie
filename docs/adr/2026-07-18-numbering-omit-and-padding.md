# ADR：编号字段空则省略与序号不补零

2026-07-18，拷问访谈定案。

- **字段段一律空则省略**：缺值不再报「编号字段无值」，静默跳过该段。为让物料一条规则覆盖通用料/客户料（`分类.code` + `客户.code` + `-` + 序号），客户空时自然得到 `01-0001`、有客户时 `0177-0001`。代价：其他资源若在规则里引用可空字段，缺值会残号且 scope 可能漂移——靠规则配置自律，不做段级开关（曾考虑 `omit_if_empty` 默认 false，未采纳以减模型复杂度）。
- **`padding: 0` = 不补零**：原 1..12 放宽为 0..12；0 时序号按十进制原样输出。不增独立 `pad` 布尔，改动面最小。
- **段拖拽排序**：前端 SegmentsEditor 支持拖拽改序，落库仍是有序数组；改序只影响此后取号，与改规则其他内容一致。
