# ADR：功能权限与公司数据权限

2026-07-07。权限校验统一落在 Ash policies，GraphQL 等入口共享。

- **功能权限 = RBAC**：权限码 `域.资源:动作`（如 `sales.order:audit`）；支持资源级 `*`、域级 `.*`、全域 `*` 通配。默认十件套动作可增删；领域动作（审核/关闭等）由资源声明。
- **权限点不入库**：由代码派生（`permission_prefix` + `permission_actions`），库只存角色→权限码授权；界面通过权限目录枚举。通配码仅迁移种子可写，界面一律写具体码（见后续权限 ADR）。
- **数据权限 = 公司维度 fail-closed**：显式公司授权挂用户不挂角色；无授权不可见。部门范围预留未做。超级管理员用户旗标 bypass 一切检查。
