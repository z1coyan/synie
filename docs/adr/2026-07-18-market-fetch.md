# 行情拉取：双价类低频免费源 + 品种级映射

在已有行情品种/不可变价点模型上接**自动/手动拉取**。业务要盘中盯最新、日终可复盘：交易时段（含有色夜盘）约每小时写 `last`；交易日日盘收市后写 `settlement`（观测时刻锚定该交易日 15:00 上海，重试不双写）。合约月不进主数据——日终在上期所官方日数据中取组内**持仓量最大**合约的结算价；盘中用新浪主连码（`nf_CU0` 等）捷径取最新价，价点 `note` 记映射信息便于对账。

价源**映射挂品种**（`fetch_enabled` / `external_last_code` / `external_product_group`），默认定时只开上期所铜铝银；长江铜等现货指数默认关、手工补。免费优先：结算走上期所公开 `.dat`，最新价走新浪公开行情串；不引入付费行情商与 Python 运行时依赖。调度用进程内 GenServer（低频、无 Oban）；手动刷新与定时共用写入链路，权限复用 `base.market_price:create`。

**可配置不黑盒**：节奏进 `sys_setting`——定时总开关、最新价间隔仅枚举 30/60/120 分、结算自动补拉开关；运行后回写上次时间与摘要，价点页只读展示。配置权限 `sys.setting`。不开放任意 cron、交易时段表或按品种不同频率（品种 `fetch_enabled` 已够）。

被否：按合约月拆主数据；仅最新价或仅结算价单通道；全品种强接（拖死现货源）；独立价源连接表；盘中也写结算价（未出炉）；非官方源当结算权威；完整调度编辑器（误配面大）。
