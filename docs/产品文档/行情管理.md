# 行情管理

## 概述

行情管理承载工厂关心的外部参考价：维护行情品种（如沪铜、长江铜、沪铝、沪银）并低频补录时序价点，供日后报价挂钩与库存估值只读引用。挂在基础数据下，菜单口语可称期货管理，领域以「行情」为准。服务采购/销售内勤与关注盘面的业务人员。

边界：只观测、落库、被引用，不持仓、不交割、不记账；本期不接销售/采购报价挂钩，不接自动拉价任务，不接库存估值。

## 核心概念

- **行情品种**：一条稳定参考价序列主数据（如「沪铜」），不按合约月拆；全局共享。见 [术语表](../glossary.md)。
- **行情价点**：某品种在某一观测时刻的不可变价格事实；有效键为（品种, 观测时刻, 价类）。
- **价类**：结算价 / 均价 / 最新价；品种有默认价类，价点可覆盖。
- **来源类型**：交易所序列 / 现货指数 / 其他。
- **取价**：≤ 目标时点的最近有效（未作废）价点；本期仅后端门面，界面不暴露。
- **停用**：品种退出主路径的方式；有价点时不可物理删除。

## 功能清单

### 行情品种维护

- 新增、编辑、停用/启用、删除（仅无价点时可删）。
- 必填：编码、名称、来源类型、默认价类、币种、计量单位；启用默认 true。
- 编码、币种、单位、来源类型创建后不可改（钉死序列口径）；名称、默认价类、启用、备注可改。
- 编码全局唯一。
- 种子预置：沪铜（SHFE_CU，结算价）、长江铜（CJ_CU，均价）、沪铝（SHFE_AL）、沪银（SHFE_AG）；无价点。

### 行情价点维护

- 补录：选品种、观测时刻、价格；价类可空（默认品种默认价类）；币种与单位自动继承品种。
- 列表查看；不可编辑价格。
- 作废：有效价点可作废；已作废不可再作废；作废后同（品种, 时刻, 价类）可重录。
- 无物理删除入口。
- 同一有效键不可重复；价格必须大于 0。
- 录入通道字段：手工（默认）/ 拉取（为后续定时任务预留，本期 UI 固定手工）。

## 业务流程

主数据无复杂流转：先维护/确认行情品种 → 按日或按小时补录价点 → 录错则作废重录。业务单据挂钩与估值在后续任务消费取价规则。
