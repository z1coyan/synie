# 行情管理

## 概述

行情管理承载工厂关心的外部参考价：维护行情品种（如沪铜、长江铜、沪铝、沪银）、补录/拉取时序价点，并在同一页面用图表盯多品种走势，供日后报价挂钩与库存估值只读引用。挂在基础数据下，菜单口语可称期货管理，领域以「行情」为准。服务采购/销售内勤与关注盘面的业务人员。

边界：只观测、落库、被引用，不持仓、不交割、不记账；不接销售/采购报价挂钩与库存估值（规划中）。不做实时推送盘口。绝对价不做跨币种/单位同图或换算；跨口径对比仅经图区「涨跌幅」归一化模式（无量纲，见下文图区）。

## 核心概念

- **行情品种**：一条稳定参考价序列主数据（如「沪铜」），不按合约月拆；全局共享。见 [术语表](../glossary.md)。
- **行情价点**：某品种在某一观测时刻的不可变价格事实；有效键为（品种, 观测时刻, 价类）。
- **价类**：结算价 / 均价 / 最新价；品种有默认价类，价点可覆盖。
- **来源类型**：交易所序列 / 现货指数 / 其他。
- **行情拉取**：按品种映射从外部价源写入价点（来源=拉取）；启用拉取的品种才参与定时与手动刷新。
- **取价**：≤ 目标时点的最近有效（未作废）价点；业务单据点价复盘用，行情页图区不替代取价规则。
- **停用**：品种退出主路径的方式；有价点时不可物理删除。
- **行情页**：基础数据下「行情」的单一工作台——上图（多选对比）+ 下表（价点 / 品种两个维护面）。

## 功能清单

### 行情页（合并工作台）

- 菜单：基础数据 → 行情 → **行情**（唯一入口）；路由 `/base/market`。
- 布局：页头（标题、刷新行情）→ **KPI 条**（选中品种最新价与窗内涨跌）→ **走势图卡**（拉取状态摘要行含拉取设置入口、口径标识、工具行、品种多选 chips、走势图）→ **Tab**（价点明细默认 / 品种维护）。
- 深链：可用查询参数打开指定 Tab（价点 / 品种）；图区勾选与时间窗记在本机，不进地址栏。
- 图与表 **弱联动**：勾选只驱动图与 KPI，不强制改表格筛选；可提供「按当前勾选筛价点表」的快捷，清除后恢复表自身筛选。
- 权限按能力拼装（资源码仍分品种 / 价点两套，不合并为第三套）：
  - 菜单可见：能读品种 **或** 能读价点。
  - 图区与价点 Tab：需价点读权限；仅有品种权限时可进页但图区与价点隐藏，只见品种维护。
  - 仅有价点权限：图与价点可用，品种 Tab 隐藏；图上品种列表来自价点域的只读投影，不依赖品种维护读权限。
  - 补录 / 手动刷新 / 作废 / 品种增改删：各随原价点、品种写权限；拉取设置仍在基础设置，读写系统设置。

### 图区（多品种对比）

- **品种多选**（chips）：仅启用品种可勾；最多 6 条序列。是否限同口径随「对比模式」：绝对价模式勾选集合必须同一币种且同一计量单位，否则拒绝并提示；涨跌幅模式无量纲，可跨币种/单位混选（如沪银元/千克与元/吨的铜铝同图）。
- **对比模式**：**绝对价**（默认，纵轴为价格，限同口径）/ **涨跌幅**（纵轴为窗内涨跌幅 %，各序列以窗内首个有效点为 0% 基准，跨口径可同图；0% 虚线参考；tooltip 同时给出涨跌幅与原始价）。切回绝对价时跨口径勾选自动收敛到首个品种所在口径组并提示。涨跌幅跨口径取数按（币种, 单位）分组并发请求时序查询再前端合并，后端同口径校验不变。
- **默认勾选**：启用品种中，按（币种, 计量单位）分组，取品种数最多的一组全部勾选（并列时排序稳定取一组），再截断到上限 6。打开时校验本机已记勾选，失效项丢弃并必要时回落默认。
- **价类**：整图统一切换（结算价 / 最新价 / 均价）；默认结算价。某品种在该价类无点则该序列空缺，**不**用其他价类冒充。
- **时间范围**：近 7 日 / 近 30 日（默认）/ 近 90 日 / 近 1 年，以及自定义起止日；按观测时刻闭区间，仅未作废价点入图。
- **KPI 条**：随当前勾选变化；展示该价类、该时间窗内末点价格，以及相对窗内首点的涨跌。与图同叙事，不引入独立「昨收」字段。
- **走势图**：多序列折线（HeroUI 图表）；无数据时明确空态。
- **取数**：图与 KPI 使用专用时序只读查询（多品种 × 价类 × 时间窗），与下方列表分页维护分离；作废点永不入图。手动刷新成功后图与列表一并更新。
- **本机状态**：记住勾选品种、价类、对比模式、时间档或自定义区间；不跨设备、不进 URL。

### 行情品种维护

- 位于行情页「品种维护」Tab；能力与独立页时期等价。
- 新增、编辑、停用/启用、删除（仅无价点时可删）。
- 必填：编码、名称、来源类型、默认价类、币种、计量单位；启用默认 true。
- 编码、币种、单位、来源类型创建后不可改（钉死序列口径）；名称、默认价类、启用、备注可改。
- 拉取映射可改：是否启用拉取、外部最新价代码（如 `CU0`）、外部品种组（如 `cu`）。
- 编码全局唯一。
- 种子预置：沪铜（SHFE_CU，元/吨，结算价，拉取开）、长江铜（CJ_CU，元/吨，均价，拉取关）、沪铝（SHFE_AL，元/吨）、沪银（SHFE_AG，**元/千克**，上期所口径）；无价点。

### 行情价点维护

- 位于行情页「价点明细」Tab（默认 Tab）；能力与独立页时期等价。
- 补录：选品种、观测时刻、价格；价类可空（默认品种默认价类）；币种与单位自动继承品种。
- 列表查看；不可编辑价格。
- 作废：有效价点可作废；已作废不可再作废；作废后同（品种, 时刻, 价类）可重录。
- 无物理删除入口。
- 同一有效键不可重复；价格必须大于 0。
- 录入通道：手工 / 拉取。

### 行情刷新

- **定时**（可配置）：在「基础数据 → 基础设置 → 行情拉取」维护——**启用定时拉取**、**最新价间隔**（30/60/120 分钟）、**启用日终结算自动补拉**。默认：定时开、间隔 60 分、结算补拉开。
  - 定时开时：交易时段（日盘约 09:00–15:00、有色夜盘约 21:00–次日 02:30，上海时区）按间隔拉启用品种的**最新价**；结算补拉开时，工作日 15:30 起尝试**结算价**（失败 16:00/16:30/17:00 重试；已有当日有效结算点则跳过）。
  - 定时关：仅手动刷新。
- **可见性**：行情页展示当前规则摘要与**上次运行时间/结果**；设置页可改旋钮并看完整上次摘要。不开放自由 cron 或交易时段表。
- **手动**：行情页「刷新行情」——对启用品种立即拉最新价；若已过日盘结算窗口且结算自动补拉开启、当日尚无有效结算点则顺带补结算。
- 日终结算口径：上期所公开日数据中，品种组内持仓量最大合约的结算价；盘中最新价走主连公开行情。
- 权限：手动刷新复用行情价点录入权限；拉取配置读写系统设置。

## 业务流程

主数据：先维护/确认行情品种（交易所序列可开拉取并填映射）→ 定时或手动刷新写入价点，必要时手工补录 → 在行情页勾选同口径品种看走势与 KPI → 录错则作废重录。业务单据挂钩与估值在后续任务消费取价规则。
