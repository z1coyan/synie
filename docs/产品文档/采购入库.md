# 采购入库

## 概述

采购入库模块承载对供应商/内部公司的**履约入库**：从已审核采购订单条目分批收货入库，审核后增加库存、累加订单已收数量，有金额时按头上借贷科目确认未开票应付。服务采购内勤与仓管。

边界：不做存货估值/进项税拆分（税在发票、库存分录只记数量，估值整体留跟进）；内部公司对手只记买方账，不自动镜像卖方出库；无反审核/红冲；入库与发票勾稽未实现。

## 核心概念

- **采购入库单**：履约入库单据，状态机 草稿→已审核→（已作废）。见 [术语表](../glossary.md)。
- **入库条目**：必挂订单条目；行仓必填；数量折默认单位；行保存冻物料与订单条目快照（含图纸挂接复制）。
- **已收数量**：订单条目上默认单位累计，审核加、作废减。
- **入库超收比例**：供应链设置全局容差，默认 0%（禁超收）；入口「供应链 → 设置 → 供应链设置」。
- **物料快照**：行保存时冻结编号/名称/规格/客户料号/单位名，并把物料当前图纸（`drawing` 槽位）挂接复制到行；审核锁行即冻结，主数据后续变更不回溯。图纸为挂接复制（共享不可变文件，非字节复制），行/入库单删除时挂接一并清理。口径与订单条目快照一致，见 [术语表](../glossary.md)。

## 功能清单

### 列表视图

- 双 tab：**入库条目**（默认）与**入库单**；共用入库单抽屉。
- **入库条目** tab 展示行快照口径的物料信息，另有图纸列（缩略图，点开可预览该行全部图纸）。
- 菜单：供应链 → 库存 → 采购入库。

### 入库录入

- 头：公司（建后不可换）、入库单号（可留空取号）、入库日期、过账日期、对手（供应商/内部公司）、默认仓库（可空，新建行预填）、借方科目（自选）、贷方科目（未开票应付角色）、备注。
- 行：订单条目（须已审核订单）、数量、单位（可空=订单单位）、入库仓库（必填）、行备注；保存时同步冻结物料快照与图纸挂接。
- 可跨多张订单，但公司、对手、原币须一致。

### 审核

- 至少一行；行仓须本公司启用叶子仓。
- 超收校验：累计已收 ≤ 订购 base × (1 + 超收比例)。
- 写正向库存分录；有金额时写总账两行（贷未开票应付、借选定科目）；累加已收数量。
- 整单金额为零：跳过总账，科目可空。

### 作废

- 仅已审核可作废：库存/总账分录作废，回滚已收数量（作废减库存，仍过负库存校验）。

### 与订单交叉

- 关闭订单后不可再入库；有已审核入库时订单不可作废；草稿入库须先删再作废订单。

## 业务流程

录入草稿 → 审核算库存与应付 → 纠错作废重开。订单部分收完可关闭剩余。
