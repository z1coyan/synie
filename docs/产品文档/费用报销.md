# 费用报销

## 概述

员工垫付费用的报销，分两步走：第一步，财务把员工交来的增值税发票登记为「费用报销」类型发票，审核即完成挂账——公司欠员工的钱成立，进应收应付报表；第二步，创建报销单给员工打款，核销欠款。五金店现金采购这类无票非税支出不经过发票，直接在报销单上手工填行。本模块不覆盖报销审批流、部分付款、报销冲抵员工借款与进项税抵扣认证。

## 核心概念

- **费用报销发票**：增值税发票第四类型，对手限员工，审核即挂账（详见 [发票](./发票.md)）。
- **报销单**：对员工付款的核销单据，状态 草稿 → 已审核 →（已作废）。
- **挂票行**：报销单引用一张该员工「已审核未报销」发票的行。
- **无票行**：报销单上手工填写的非税支出（摘要 + 金额 + 报销类型）。
- **报销类型**：即费用类科目角色（差旅费 / 办公费 / 业务招待费 / 交通费 / 其他费用），选定后自动带出费用科目。
- **其他应付款**：员工报销挂账专用的科目角色；员工欠款计入应收应付报表的净应付。

## 功能清单

### 报销发票登记与挂账

- 新增发票选「费用报销」类型：对手限员工，无需关联对账单；选报销类型后自动带出金额科目与往来科目（取本公司挂对应角色的唯一启用科目，缺挂或多挂时报错改手选），税额科目手选。
- 审核过账同其他开入发票：借 金额科目（未税金额）、借 税额科目（税额）、贷 往来科目（价税合计，带员工对手）。过账即完成挂账，不生成任何下游单据。
- 发票一旦被报销单挂票行引用（含草稿报销单）即锁定，不可作废、不可红冲；须先在报销单草稿上移除该行，或作废整张报销单。

### 报销单（付款核销）

- 单头：公司、员工、报销日期、过账日期、付款科目（贷方，银行存款 / 库存现金类）、单号（留空自动取号）、备注。
- 挂票行：从本公司、该员工名下「已审核未报销」的费用报销发票中挑选；一张发票全额核销、只能进一张报销单，不支持部分付款。
- 无票行：手填摘要、金额、报销类型（自动带费用科目）；无票支出没有税额。
- 审核过账：借 各挂票行发票的往来科目（价税合计，带员工对手）、借 各无票行费用科目（行金额），贷 付款科目（全部行合计）。
- 已审核报销单只能作废（无红冲）：原分录标记作废，同时解除全部发票引用，发票回到「已审核未报销」，可被新报销单重新引用；纠错 = 作废 + 重开。
- 草稿可自由修改、删除。

## 业务流程

员工陆续交票 → 财务逐张登记（可 OCR）报销类型发票并审核，应收应付报表出现该员工欠款 → 集中付款时创建报销单：挑选本次要付的发票、补录无票支出 → 审核过账，欠款核销。无票支出不需要任何前置单据。
