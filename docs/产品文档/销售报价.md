# 销售报价

## 概述

销售报价模块承载公司对客户（或集团内其他公司）的价格承诺：按物料+单位报出含税价格，支持一口固定价与按订购量分档的数量梯度价，审核后报价生效锁死，到截止日自然失效。服务销售内勤与报价决策人员。

边界：报价单是纯价格清单——条目不含数量，不构成供货承诺，审核不派生任何会计或库存后果；报价转销售订单、期货梯度报价（挂钩交货期或期货盘面）均未实现，后者已在数据模型上预留扩展位。

## 核心概念

- **销售报价单**：对客户/内部公司的价格承诺单据，状态机 草稿→已审核→已作废。见 [术语表](../glossary.md)。
- **报价条目**：报价单的价格行，记录物料、单位、定价模式、含税单价、税率、行备注；同一报价单内（物料, 单位）唯一，同物料不同单位可并存（两种成交口径）。
- **定价模式**：逐条目选择——**固定价**（行上一口含税价）或**数量梯度**（行单价空置，价在价格档上）。
- **价格档**：数量梯度条目的起量价阶梯，每档 =（起订量, 含税档价），订购量达到起订量即适用该档价；低于首档起订量视为无报价。
- **报价截止与已过期**：截止当日仍有效；已审核单过了截止日在列表中显示为「已过期」（派生展示态，数据库状态仍是已审核，仍可作废）。
- **币种**：一单一币挂报价单头，默认公司本币；报价单无金额，不涉及汇率与双币换算。
- **对手**：报价的收方，取客户主数据或内部公司（多态引用，同销售订单）。

## 功能清单

### 列表视图

- 页面分两个 tab（子路由，URL 可直达）：**报价条目**（默认视图，行级明细——每行展示所属报价单号、报价日期、报价截止、对手、状态，以及行快照口径的物料编号/名称/规格、单位、币种、定价模式、含税单价（梯度行留空）、档数、行备注；默认按报价日期倒序；点报价单号或行「查看」开完整报价抽屉，草稿单可经行「编辑」直接改单）与**报价单**（整单列表，含审核/作废/删除动作）。
- 两个 tab 共用同一个报价抽屉（新增/查看/编辑三态），「新建报价单」入口两个 tab 都有；审核/作废/删除只在报价单 tab。
- 状态胶囊：草稿灰、已审核绿、已作废红；已审核且过截止日显示黄色「已过期」。

### 报价录入

- 头：报价单号、报价日期（默认当天）、报价截止、公司、对手、币种、报价条款（对客户，自由多行文本）、报价备注（对内）。
- 报价单号留空按编号规则自动取号，手填原样保留；报价单号全局唯一。
- 报价截止不得早于报价日期（同日允许）。
- 对手限客户/内部公司两类；选内部公司时不能是报价单所属公司自身。
- 币种：选公司后自动带出公司本币，可改选其他币种；同一物料要报多个币种时开多张报价单。
- 条目行：物料、单位、定价模式、含税单价（固定价）、税率、行备注；单位只能选该物料的默认单位或其单位转换单位，选物料后默认带默认单位。
- 定价模式选「数量梯度」时，行表单内出现价格档子表：逐档录入起订量与含税档价；起订量必须大于零且同条目内唯一，档价可为零；条目表上梯度行以「≥量 价 / …」阶梯概要展示。
- 条目从数量梯度切回固定价时，其价格档全部清空，需重新填写含税单价。
- 条目约束：固定价条目必须填含税单价（可为零）；税率必填，界面按百分比录入（默认 13%，存储为小数），仅作口径标注不参与计算。

### 审核

- 仅草稿单可审核；审核前必须至少有一行条目，且每个数量梯度条目至少有一个价格档。
- 审核后单据锁死：头不可改、行与价格档不可增删改、单不可删；无反审核。

### 作废

- 仅已审核单可作废（含已过期单），终态不可逆；承担录入错误与提前撤回报价（如市场价变动）两种场景。

### 删除

- 仅草稿单可删除，条目与价格档随单一并删除。

## 业务流程

录入（草稿，可反复改）→ 审核（生效，可对外发出）→ 到截止日自然过期；截止前要撤回报价则「作废」。报价延期没有改截止日的动线，重新开一张新报价单。
